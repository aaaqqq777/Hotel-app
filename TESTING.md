# 前端项目测试文档

## 测试概述

本项目包含全面的前端测试，覆盖搜索页、列表页和详情页的功能性与自适应能力。

## 测试分类

### 1. 搜索页测试 (`homepage.test.tsx`)
- 组件渲染测试
- 表单交互测试
- Tab 切换测试
- 响应式布局测试

### 2. 列表页测试 (`hotellist.test.tsx`)
- 酒店列表渲染
- 筛选功能测试
- 排序功能测试
- 无限滚动测试
- 筛选状态保持测试

### 3. 详情页测试 (`detailpage.test.tsx`)
- 酒店详情渲染
- 图片轮播测试
- 房型预订功能
- 设施展示测试
- 响应式布局测试

### 4. 自适应能力测试 (`responsive.test.tsx`)
- 移动设备适配
- 平板设备适配
- 桌面设备适配
- 屏幕旋转测试
- 跨设备一致性测试

## 运行测试

```bash
# 运行所有测试
npm run test:run

# 运行测试并生成覆盖率报告
npm run test:run -- --coverage

# 以交互模式运行测试
npm run test
```

## 测试覆盖率目标

- 组件渲染: 100%
- 用户交互: 95%+
- 边界情况: 90%+
- 响应式设计: 100%

## 测试策略

### 功能测试
- 验证各页面核心功能
- 检查用户交互流程
- 确保数据正确显示

### 稳定性测试
- 组件生命周期测试
- 状态管理测试
- 错误边界测试

### 自适应测试
- 不同屏幕尺寸测试
- 设备方向变化测试
- 响应式布局验证
