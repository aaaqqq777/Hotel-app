# TripCom 用户使用手册

## 应用概述

TripCom 是一款现代化的酒店预订移动应用，为用户提供便捷的酒店搜索、浏览和预订服务。

## 快速开始

### 系统要求
- **操作系统**: iOS 13.0+ / Android 6.0+
- **浏览器**: Chrome 80+、Safari 13+、Edge 80+
- **网络连接**: 稳定的互联网连接

### 访问应用
- **开发环境**: http://localhost:5173
- **生产环境**: 根据部署地址访问

## 功能使用指南

### 首页（搜索页）

**核心功能**：
1. **搜索表单**：填写城市、入住日期、退房日期、房间数和人数
2. **Tab 切换**：国内酒店、海外酒店、钟点房、民宿
3. **城市选择**：点击城市输入框选择目的地
4. **日期选择**：点击日期输入框选择入住和退房日期
5. **标签筛选**：选择酒店特色标签
6. **广告横幅**：查看推荐酒店广告

**操作步骤**：
1. 选择入住城市
2. 选择入住和退房日期
3. 设置房间数量和入住人数
4. 选择酒店类型
5. 点击搜索按钮进入酒店列表页

### 酒店列表页

**核心功能**：
1. **搜索结果**：展示符合条件的酒店列表
2. **排序功能**：默认排序、价格从低到高、价格从高到低、评分优先
3. **筛选功能**：价格区间、星级、评分
4. **快捷标签**：点击标签快速筛选酒店
5. **无限滚动**：下拉加载更多酒店
6. **酒店卡片**：展示酒店基本信息、价格、评分

**操作步骤**：
1. 浏览酒店列表
2. 点击排序按钮选择排序方式
3. 点击筛选按钮设置筛选条件
4. 点击标签快速筛选
5. 下拉页面加载更多酒店
6. 点击酒店卡片进入详情页

### 酒店详情页

**核心功能**：
1. **图片轮播**：查看酒店多张图片
2. **酒店信息**：名称、星级、评分、地址等
3. **设施标签**：展示酒店提供的设施和服务
4. **地图定位**：查看酒店位置
5. **房型列表**：查看可用房型及其价格
6. **底部操作栏**：联系酒店、查看房型
7. **评价弹窗**：查看酒店评价详情

**操作步骤**：
1. 浏览酒店图片
2. 查看酒店基本信息
3. 点击地图位置查看详细地址
4. 浏览可用房型
5. 点击底部按钮联系酒店或查看房型
6. 点击评分区域查看评价详情

## 常见操作

### 搜索酒店
1. 在首页填写搜索条件
2. 点击搜索按钮
3. 在列表页查看搜索结果
4. 根据需要调整筛选条件

### 预订酒店
1. 在酒店详情页查看房型信息
2. 选择合适的房型
3. 点击预订按钮
4. 填写预订信息
5. 完成支付

### 联系酒店
1. 在酒店详情页底部点击电话按钮
2. 在弹出的电话弹窗中选择拨打电话或复制号码
3. 根据提示完成操作

### 查看地图
1. 在酒店详情页点击地图位置
2. 自动打开高德地图应用或网页版
3. 在地图中查看酒店位置和路线

## 故障排除

### 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 搜索无结果 | 搜索条件过窄 | 放宽筛选条件或更换搜索词 |
| 图片加载失败 | 网络连接问题 | 检查网络连接或刷新页面 |
| 地图无法显示 | 定位权限问题 | 允许浏览器获取位置权限 |
| 页面加载缓慢 | 网络速度慢 | 检查网络连接或稍后重试 |
